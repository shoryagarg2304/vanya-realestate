# Vanya Real Estate Website

## How to deploy to Vercel (no technical knowledge needed)

### Option A – Drag & Drop (Easiest, no account needed initially)
1. Go to https://vercel.com
2. Click "Sign Up" → use your Google or GitHub account
3. After signing in, look for "Add New Project" or just drag your project folder onto the page
4. Vercel will detect it's a Vite/React project automatically
5. Click "Deploy" — done! You'll get a live link in ~1 minute.

### Option B – Via GitHub (Recommended for future updates)
1. Create a free account at https://github.com
2. Create a new repository and upload this entire folder
3. Go to https://vercel.com and sign in with GitHub
4. Click "Add New Project" → select your repository
5. Click Deploy → live link in ~1 minute

Your site will be live at something like: https://vanya-realestate.vercel.app
